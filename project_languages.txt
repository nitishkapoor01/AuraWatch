AuraWatch Project - Technical Architecture & Roadmap
==================================================

1. Core Technologies
--------------------
- JavaScript (JSX): React Frontend & Node.js Backend.
- PostgreSQL: Database for Users, History, and Settings.
- CSS Modules: Scoped premium styling (Glassmorphism & Neon Glow).
- Playwright: High-speed scraper for movie link aggregation.

2. Database Logic (The "Brain")
-------------------------------
- Sticky Watch History: 'is_completed' flag ensures re-watching doesn't reset progress.
- Role-Based Access: Admin, Moderator, and User levels.
- Dynamic Settings: JSON-based key-value storage for warnings, ads, and updates.

3. Premium UI/UX System
-----------------------
- Toast Engine: Global notification system with bounce animations and pulse glows.
- Warning Badges: Animated pulse dots on restricted buttons (Play/Download).
- Responsive Design: Custom mobile-optimized layouts for Sidebar and Modals.

4. Admin Control Center
-----------------------
- Engagement Analytics: Differentiates between 'Total Clicks' and '90%+ Completed Watches'.
- Button Warnings: Remote-control alerts for restricted actions.
- Platform Updates: Real-time changelog manager visible in Help & Support.
- Live Traffic: Real-time monitoring of active users and search trends.

5. Deployment & Tools
---------------------
- Backend: Hosted on Render (Node.js/Express).
- Frontend: High-performance React (Vite).
- Git: Version control for feature branching.
- SEO Engine: Dynamic meta tags for Movie/Series discovery.
